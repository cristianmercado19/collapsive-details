Project Instructions

This project enables users to create dynamic HTML emails with interactive {} placeholders that can be expanded or collapsed for detailed information as needed. The final email can be securely encrypted and shared via GitHub Gist, allowing recipients to decrypt and explore the content interactively using a provided URL and password.

Example of the Email:

Hi,
We are reminding you of your upcoming appointment on Friday { 
  Appointment Details:
- Time: 10.30
- Location: Dublin Office
}

Remember to bring your documentation { You will need to bring a 
  valid ID and your insurance card. Contact Info {If you have 
    any questions or need to reschedule, please contact us at}.}

We look forward to seeing you soon.

Best regards,

Steps to Follow:

  1. Generate HTML:
    • Click the "Generate" button.
    • Review the generated HTML to ensure it displays correctly and supports user interactions.
  2. Encrypt the HTML:
    • Enter a decryption key (password) in the input field.
    • Click the "Encrypt" button to generate the encrypted text.
  3. Store Encrypted HTML in GitHub Gist:
    • Copy the encrypted text.
    • Go to GitHub Gist.
    • Create a new Gist, paste the encrypted text, and save it.
    • Copy the generated Gist URL.
  4. Share the URL and Password:
    • Provide the GitHub Gist URL and the decryption password to the intended recipient.
  5. Recipient View the Email:
    • The recipient opens the Collapsive Details viewer specifically developed for GitHub Gist support.
    • They enter the GitHub Gist URL and the decryption password.
    • The email content is decrypted and displayed, allowing the recipient to interact with it.
